# AI Assistant 合作伙伴白皮书

---

## 产品概述

AI Assistant 是一款基于大模型的企业级智能客服系统，支持多租户 SaaS 部署，为合作伙伴提供快速交付智能化客服解决方案的能力。

## 为什么选择合作

| 优势 | 说明 |
|------|------|
| 🛠️ 快速部署 | Docker 一键部署，30 分钟内上线 |
| 📦 标准化产品 | 功能模块化，易于二次开发 |
| 🔌 开放 API | 完整 RESTful API，支持集成 |
| 💼 多租户 | 支持 SaaS 模式，合作伙伴可自运营 |
| 🌍 国产化 | 兼容 MiniMax 等国产大模型 |

## 技术架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   前端 UI   │ ←→  │  FastAPI   │ ←→  │  大模型 API │
│  (HTML/JS)  │     │  (Python)  │     │ (MiniMax)   │
└─────────────┘     └─────────────┘     └─────────────┘
                           ↓
                    ┌─────────────┐
                    │  SQLite +   │
                    │   Chroma    │
                    └─────────────┘
```

### 技术栈
- 后端：FastAPI (Python)
- 前端：原生 HTML/CSS/JS
- 数据库：SQLite + Chroma 向量库
- 模型：MiniMax / OpenAI 兼容

## 核心功能模块

### 多租户 SaaS
- 租户管理、用户管理、用量统计
- 合作伙伴可自建租户运营

### 对话系统
- RAG 知识库问答
- 提示词模板
- Function Calling 工具

### 客服系统
- AI ↔ 人工无缝切换
- 客服状态管理
- 会话分配与转移

### 管理后台
- 会话管理（搜索/导出/置顶）
- 关键词规则引擎
- 企业微信集成
- 角色权限控制

## 合作模式

### 1. 代理合作
- 合作伙伴负责销售与实施
- 提供技术支持和产品培训

### 2. OEM 定制

- 定制功能开发

### 3. SaaS 运营
- 合作伙伴自建多租户平台
- 按租户/用量收费

- 自定义前端界面

**API 覆盖：**
- 认证、用户、租户管理
- 会话、消息、知识库
